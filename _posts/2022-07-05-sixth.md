## MERGE
- 분산되어 있던 대량의 데이터를 통합하여 분석하기 위한 업무에 유용하게 사용된다.
- MERGE의 사용법은 아래와 같다.
```
MERGE INTO [table] [alias]
USING [table / view / subquery] [alias]
ON [join condition]
WHEN MATCHED THEN
    UPDATE SET ......
WHEN NOT MATCHED THEN
    INSERT INTO ......
    VALUES ......;
```
- MERGE INTO : 하나의 테이블로 합치기 위한 결과 테이블을 정의한다. 즉, [table]에 기록된 테이블에는 합쳐진 결과가 기록된다.
